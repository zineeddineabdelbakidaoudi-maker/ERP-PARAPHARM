## 📖 About The Project

**ParaFarm ERP** is a comprehensive, desktop-based Enterprise Resource Planning (ERP) application engineered to streamline the daily operations of pharmacies and parapharmacies. Built with robust Python technologies and a sleek PySide6 graphical interface, it simplifies complex inventory tracking, sales processing, and supplier debt management.

> *Note: For security and proprietary reasons, some core logic files and the database are intentionally excluded from this repository. This repository serves as a showcase of the software's architecture and capabilities.*

## ✨ Key Features

- **🛒 Point of Sale (POS):** Lightning-fast checkout, barcode scanning support, and automatic stock deduction.
- **📦 Inventory Management:** Real-time tracking of medicinal stocks, expiration date alerts, and reordering workflows.
- **📈 Advanced Reporting:** Detailed sales analytics, PDF invoice generation, and revenue charts.
- **🤝 Supplier & Debt Tracking:** Manage supplier invoices, track outstanding debts, and process partial payments seamlessly.
- **🖨️ Thermal Printing:** Direct integration with ESC/POS thermal printers over USB and Network, with a reliable PDF fallback.

## 🛠️ Technology Stack

- **Backend:** Python 3.12
- **Frontend / UI:** PySide6 (Qt6 Framework)
- **Database:** SQLite (managed via SQLAlchemy ORM)
- **Migrations:** Alembic
- **Document Generation:** ReportLab (PDF generation)

## 💡 The Notion Behind ParaFarm

The pharmaceutical retail space requires absolute precision. Stock discrepancies or delayed checkouts can negatively impact customer health and business operations. ParaFarm was designed with the notion of **"Frictionless Pharmacy Operations"**. By centralizing POS, inventory, and supplier debts into a single, intuitive dashboard, pharmacists can focus more on patient care and less on administrative overhead.
